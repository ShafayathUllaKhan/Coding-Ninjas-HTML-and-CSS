2. transform 
enables us to rotate, scale, skew or translate any element

2d tranform function values can be given as 

none
rotate rotatex rotatey
scale scalex scaley
skew skewx skewy
translate translatex translatey

Animation is a shorthand for the following properties

Animation-name
Animation-duration
Animation-timing-function
Animation-delay
Animation-iteration-count
Animation-direction
Animation-fill-mode
Animation-play-state
